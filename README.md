# Personal Finance Tracker (Python)

this is a simple command line in python wich aims to track peronal financial especifically incomes and expenses
This project was created to practice core programming concepts and demonstrate clean, object-oriented design.

## Features

- Add income Transactions
- Add expense transaction
- View current balance
- Input Validation and error handling

## Concepts Used

- Object - Oriented Programming (OOP)
- Classes and instances
- Lists and dictioaries
- Control flow (if / elif / else)
- Exception handling (try, except)
- Modular project structure
